# Star Skill · 信息录入向导

> 当用户输入 `/create-star` 时，运行本向导。

---

## 第一步：人物基本信息

请提供以下信息（可跳过，仅描述也可）：

| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名/艺名 | 真实姓名 + 艺名 | 王澳楠 / EVE |
| 平台账号 | B站/微博/网易云等 | B站: 85841036 |
| 生日 | YYYY-MM-DD | 1999-12-20 |
| MBTI | 16型人格 | ENFP |
| 职业 | 歌手/演员/网红等 | 说唱歌手 |
| 代表作 | 标志作品 | 《逐客令》《让他走》 |
| 平台签名 | 社交媒体签名档原文 | 「大家好我是EVE！你的音乐止痛药！」 |

**或直接用一段话描述：**
```
「王澳楠，1999年12月20日生，ENFP说唱歌手，B站/微博「王澳楠EVE」，
代表作《逐客令》，MBTI是ENFP，人设是音乐止痛药，甜丧风格，
B站签名「大家好我是EVE！你的音乐止痛药！」」
```

---

## 第二步：数据来源选择

**A. 全自动采集**（推荐，有API的平台）
```
□ 网易云音乐 - 歌词（需提供歌手ID）
□ B站 - 视频详情+评论（需提供UID）
□ 微博 - 动态（需提供UID）
```

**B. 半自动采集**
```
□ 手动上传 JSON/TXT 文件
□ 粘贴歌词/微博文字
□ 截图 + OCR
```

**C. 仅描述**（质量较低，但可行）
```
□ 纯文字描述：描述她的说话风格、性格、代表作等
```

---

## 第三步：补充描述

请描述她的**性格特点**、**说话风格**、**标志性口头禅**、**粉丝互动方式**等。

可以参考以下维度：
- 她生气时怎么说？
- 她开心时怎么说？
- 她深夜发微博是什么风格？
- 她怎么称呼粉丝？
- 她的核心金句有哪些？
- 有什么禁忌话题？
